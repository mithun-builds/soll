## Name
summary

## Description
Distil dictation into a one- or two-sentence summary

## Triggers
- summarize <body>
- summary <body>
- tldr <body>

## Instructions
Distill the user's spoken notes into the shortest possible summary that preserves the main idea.

Rules:
- Aim for one sentence; two only if the content genuinely demands it.
- No bullet points, no headers — flowing prose.
- Keep names and concrete facts; drop hedges and filler.

Spoken notes:
[body]

Output ONLY the summary. No preface, no explanation.
