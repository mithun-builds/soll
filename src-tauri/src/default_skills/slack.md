## Name
slack

## Description
Format dictation as a casual Slack message

## Triggers
- slack message <body>
- send a slack <body>
- as a slack <body>

## Instructions
Take the user's spoken notes and rewrite them as a short, casual Slack message.

Rules:
- No greeting, no sign-off — Slack messages don't need them.
- Casual tone is fine; avoid formal "Dear" / "Best regards" registers.
- Use line breaks between separate points; one idea per line.
- Keep it under three sentences when possible.

Spoken notes:
[body]

Output ONLY the Slack message. No commentary.
